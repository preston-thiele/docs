---
title: 'Quickstart'
description: 'Start building awesome documentation in under 5 minutes'
---

1. Install Danube
```bash
pip install danube
pip install openai
```

2. Get API Key from Danube dashboard

3. Set API Key in Danube
```bash
os.environ["DANUBE_API_KEY"] = "<API_KEY>"
os.environ["OPENAI_API_KEY"] = "<OPENAI_API_KEY>"
```

4. Initialize the Danube client
```python
from danube import Danube
from openai import OpenAI

danube = Danube()
openai = OpenAI()
```
5. Set the system prompts
```python
messages = [
    {"role": "system", "content": danube.system_prompt},
    {"role": "user", "content": danube.identity},
    {"role": "user", "content": "What are the top 5 stories on Hacker News?"},
]
```
6. Provide your model with Danube tools
```python
response = openai.chat.completions.create(
            model="gpt-4o",
            messages=messages,
            tools=danube.tools,
            tool_choice="auto",
            max_tokens=4000
        )
```
